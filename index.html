동현랭 소개

동현랭은 한글로 프로그래밍할 수 있는 간단한 프로그래밍 언어입니다. 기본적인 프로그래밍 개념을 한글로 배울 수 있도록 설계되었습니다.

문법

변수 할당

봑 변수명 = 값 : 변수를 선언하고 값을 할당합니다.

봑 number = 10
봑 text = "안녕하세요"

조건문

동현? 조건식 : 조건이 참일 때 실행할 코드 블록을 시작합니다.

동?현 조건식 : 이전 조건이 거짓일 때, 새로운 조건이 참일 때 실행할 코드 블록을 시작합니다.

?동현 : 이전의 모든 조건이 거짓일 때 실행할 코드 블록을 시작합니다.

동현? number == 10
    현동("number는 10입니다.")
동?현 number == 5
    현동("number는 5입니다.")
?동현
    현동("number는 10도 5도 아닙니다.")

반복문

똥똥현현 조건식 : 조건이 참일 때까지 반복해서 실행합니다.

똥똥현현 number > 0
    현동(number)
    봑 number = number - 1

함수 정의 및 호출

바아도오 함수명(매개변수들) : 함수를 정의합니다.

도오 함수명 : 함수를 호출합니다.

형돈 반환값 : 함수를 통해 값을 반환합니다.

바아도오 환영메시지()
현동 "안녕하세요"
도오 환영메시지()
